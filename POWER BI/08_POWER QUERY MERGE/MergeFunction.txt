MERGE IS SIMILAR TO VLOOKUP IN EXCEL THAT BRINGS COLUMNS FROM OTHER TABLE AS WELL

1. MERGE TABLES FROM A SINGLE EXCEL FILE
	SELECT TABLES -> MERGE QUERIES AS NEW -> SELECT TABLES
	-> SELECT MATCH ROWS -> OK
TRY TO CHANGE THE JOINS.

2. MERGE 2 DIFFERENT EXCEL FILE IS SAME AS ABOVE

3. MERGE TABLES FROM 2 DIFFERENT SOURCES SAME AS ABOVE

4. MERGE DIFF COLUMNS -> MERGE QUERIES -> 
	CTRL+SELECT COLS TO MATCH
